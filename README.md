# Abjad Arabic Workbook

A project for generating Arabic language workbooks focused on the Abjad script and letter forms.

## Project Structure

- `abjad_production/` - Main production code for workbook generation
- `data/` - Data files including letter forms and illustrations metadata
- `templates/` - SVG templates for workbook pages
- `assets/` - Static assets including illustrations and fonts
- `output/` - Generated output files (SVG, PNG, PDF)

## Getting Started

Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Generate workbook:
```bash
python abjad_production/generate_workbook.py
```
